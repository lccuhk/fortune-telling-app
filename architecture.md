# 算命应用系统架构

## 整体架构

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│                 │     │                 │     │                 │
│   前端应用       │────▶│   后端 API      │────▶│   模型服务      │
│  (Vue + Vite)   │◀────│  (FastAPI)      │◀────│  (语言模型)     │
│                 │     │                 │     │                 │
└─────────────────┘     └─────────────────┘     └─────────────────┘
```

## 详细架构

### 1. 前端层
- **技术栈**：Vue 3 + Vite
- **主要功能**：
  - 用户界面交互
  - 表单提交（姓名、出生日期、问题）
  - 结果展示
  - 响应式设计

### 2. 后端层
- **技术栈**：FastAPI
- **主要功能**：
  - API 接口提供
  - 请求处理
  - 数据验证
  - CORS 配置
  - 错误处理

### 3. 数据科学管道
- **数据收集**：
  - 示例数据加载
  - 用户输入数据
- **数据预处理**：
  - 数据格式验证
  - 数据清洗
  - 数据标准化
- **模型集成**：
  - 基于规则的预测
  - 语言模型集成（预留接口）

### 4. 模型层
- **当前实现**：基于规则的预测系统
- **未来扩展**：集成 Seed-Dogfooding-2.0 语言模型
- **预测逻辑**：
  - 根据问题类型生成不同的预测
  - 随机选择预测结果和建议

## 数据流

1. **用户输入**：用户在前端填写姓名、出生日期和问题
2. **数据传输**：前端通过 POST 请求将数据发送到后端
3. **数据处理**：后端对输入数据进行预处理和验证
4. **模型预测**：调用模型生成预测结果和建议
5. **结果返回**：后端将结果返回给前端
6. **结果展示**：前端展示预测结果和建议

## 技术选型理由

1. **前端**：Vue 3 + Vite
   - 轻量级框架，性能优异
   - 响应式设计，开发效率高
   - 生态成熟，文档丰富

2. **后端**：FastAPI
   - 高性能异步框架
   - 自动生成 API 文档
   - 类型提示，代码质量高
   - 易于集成机器学习模型

3. **数据处理**：Python
   - 数据科学生态丰富
   - 易于集成各种模型
   - 开发效率高

4. **模型**：基于规则 + 预留语言模型接口
   - 快速实现原型
   - 可扩展性强
   - 未来可无缝集成高级语言模型
