## [2.0.0] - 2024-06-01

### 新增
- 全新 Vue 3 + Vite 技术栈重构
- 多主题切换功能（古风、科技风、现代风）
- 八字算命功能
- 塔罗牌占卜功能
- 星座运势功能
- 生肖运势功能
- 周公解梦功能
- 黄历查询功能
- 姓名分析功能
- 紫微斗数功能
- 合盘分析功能
- 用户系统（登录、注册）
- 历史记录保存
- 分享卡片生成
- 响应式设计，支持移动端
- 深色模式支持
- 3D 银河背景动画

### 改进
- 优化算命算法，提高准确度
- 改进用户界面，提升用户体验
- 优化性能，加快页面加载速度

### 技术栈
- Vue 3 + Composition API
- Vite 构建工具
- Pinia 状态管理
- Vue Router 路由
- TypeScript 类型支持
- Vitest 单元测试
